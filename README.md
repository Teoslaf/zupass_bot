# zupass_bot

Script to get the frogs.

## How to use:

### 1. Create and activate the virtual environment:

```sh
python3 -m venv myenv
source myenv/bin/activate
```
```sh
pip install selenium
```
### 2. Run:
```sh
python frog.py
```
### 3. Input your ZuPass email and password when prompted.
